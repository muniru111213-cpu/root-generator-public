# root-generator

## 概要
root-generatorは、出発地、目的地、寄り道地点を入力すると、Google MapsやApple Mapsでルートを生成し、直接地図アプリを開くことができるシンプルなアプリです。

## 使い方
1. 必要なライブラリをインストールする:
   ```bash
   npm install
   ```

2. 開発サーバーを起動する:
   ```bash
   npm start
   ```
   アプリはローカルのブラウザで自動的に開きます。

3. 入力項目:
   - **出発地**: 出発地点の住所や名前を入力します。
   - **目的地**: 到着地点の住所や名前を入力します。
   - **寄り道**: 立ち寄りたい場所を追加します（複数追加可能）。

4. 「Google Mapsで開く」または「Apple Mapsで開く」ボタンをクリックして地図アプリを起動します。

## 注意事項
- 地点を入力する際、日本語やスペースが含まれている場合でも正確に動作するように自動エンコードしています。
- すべてのフィールドを入力しないとルート生成ボタンが動作しません。

## デプロイ
1. ビルド:
   ```bash
   npm run build
   ```
   `build`フォルダに成果物が生成されます。

2. 静的ホスティングサービス（例: Vercel, Netlify）にフォルダを配置します。
